# Backtest Results

## Status

The original published figures are **invalidated**. They were generated by an engine that executed on the same close used to calculate RSI, added trade percentages instead of compounding daily returns, and calculated risk metrics from an additive trade series.

The corrected engine now:

- applies end-of-day RSI state to the subsequent return;
- compounds a daily strategy equity curve;
- applies transaction costs when position state changes;
- annualizes Sharpe from daily observations;
- calculates percentage drawdown from the running peak;
- marks an open position to the final available close.

Current numerical results must be regenerated after applying `schema.sql`, refreshing prices, and recomputing indicators:

```bash
python ingest.py
python indicators.py
python backtesting.py
```

No replacement performance claim is made until that production-data recomputation is completed.
